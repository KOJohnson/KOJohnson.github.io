Untitled Stealth Game

Inspired by Deus Ex Human Revolution and the Cyberpunk 2077.

This game allows you to complete the objective in several different by interacting with the environment to open doors, disable traps/environmental hazards and moving objects to reveal hidden paths and more!

It also features an Takedown system similar to that found in Deus Ex Human Revolution along with the special Takedown camera that is activated when such a move is performed. Interaction System

In active development is my interaction system which is also being used in my Untitled Stealth Game, to allow the player the ability to interact with their environment for a more immersive experience.

The basis of this is an Interface that defines a method Interact() and every object that implements this can easily define their own behaviour. To meet the needs of other projects I also added an InteractionContext Scriptable Object which holds additional data on the interaction that can be displayed on UI elements such as item description or the type of interaction this object has for example is it a holdable object.

All of this is then controlled by the InteractionController which uses raycasts to query the environment for objects that implement the IInteractable interface.

Cyberpunk 2077 Inspired Quick Hack System

In active development is my Cyberpunk 2077 inspired quick hack system, which is being developed alongside my Untitled Stealth Game.

With this ability the player can trigger object specific effects from great distances such as explosions which when timed correctly can take out an enemy.

The basis of this is an Interface that defines a method Hack() and every object that implements this can easily define their own behaviour. To meet the needs of other projects I also added an HackContext Scriptable Object which holds additional data on the interaction that can be displayed on UI elements such as item description or the type of interaction this object has for example is it a holdable object.

This is managed by the HackController which uses raycasts to query the environment for objects that implement the IHackable interface.

In the future I am to expand upon this system by adding cooldowns to Hack Context and also limit the amount of hacks the player can perform through the introduction of a resource similar to how quick hacks in Cyberpunk 2077 are limited by the amount of in game RAM the player has available.
